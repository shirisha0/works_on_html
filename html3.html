Drop an image file from your desktop (Finder on a Mac or Explorer on Windows) onto the VUR (Very Ugly Robot). We'll display the hidden image, if any exists. If your file does not contain a hidden image, the result will be a black and white image that probably won't look like anything! If you have a custom key image, drag that onto the key icon. Otherwise, the robot will use the default key to decode your image.
